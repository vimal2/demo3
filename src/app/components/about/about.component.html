Overview

Angular Router is a powerful routing library that enables navigation between views. It interprets browser URLs as instructions to navigate to a specific view. This demo covers everything from basic routing to advanced features like lazy loading and guards.

Features Demonstrated

  • RouterLink Directive - Declarative navigation in templates
  • RouterLinkActive - Styling active navigation links
  • Route Parameters - Dynamic URL segments (:id)
  • Query Parameters - Optional URL parameters (?key=value)
  • Fragment Navigation - Hash-based page sections (#section)
  • Programmatic Navigation - Router.navigate() method
  • Child Routes - Nested routing with router-outlet
  • Lazy Loading - Load modules on demand
  • Route Guards - Protect routes with conditions
  • Resolvers - Pre-fetch data before navigation
