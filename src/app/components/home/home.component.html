Quick Start Guide
  1. Routing: Check app.module.ts for route configuration. Visit /products to see params and nested routes in action.
  2. Guards: Try accessing /dashboard without logging in. Login with any credentials, check "Admin" to access /admin.
  3. Forms: Visit /forms to explore template-driven, reactive, and advanced form patterns.
  4. Testing: Run ng test to execute 110+ unit tests. Check *.spec.ts files for test examples.
  5. Environment Config: See src/environments/ for per-environment settings. Try ng serve --configuration=staging.
