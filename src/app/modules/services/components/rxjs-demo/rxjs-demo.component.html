RxJS Operators Cheat Sheet
Operator Purpose Common Use Case
map Transform each emitted value Convert API response to model
filter Only emit values passing condition Filter out invalid data
tap Side effects without changing stream Logging, debugging, analytics
catchError Handle errors gracefully Show error message, provide fallback
switchMap Map to new observable, cancel previous Search input, HTTP requests
mergeMap Map to observable, run in parallel Parallel HTTP requests
concatMap Map to observable, queue in order Sequential operations
debounceTime Wait for pause in emissions Search input, resize events
distinctUntilChanged Only emit when value changes Prevent duplicate API calls
takeUntil Complete when notifier emits Unsubscribe on component destroy
take Take first n emissions One-time operations
combineLatest Combine latest from multiple streams Combine filters, form values
